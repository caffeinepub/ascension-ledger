# Specification

## Summary
**Goal:** Apply a custom sci-fi hexagonal background image across the application and improve text visibility with white/near-white font colors while maintaining the dark theme.

**Planned changes:**
- Set the uploaded IMG_20260219_080023.png.jpg as a fixed background image across all application pages
- Maintain dark theme for all UI components (cards, panels, navigation) with semi-transparent dark overlays to ensure content stands out against the background
- Update all text colors to white or near-white tones (primary text: white, secondary text: rgba(255, 255, 255, 0.7)) for optimal visibility against the dark background
- Ensure the background image covers the viewport properly while maintaining aspect ratio

**User-visible outcome:** Users will see a futuristic hexagonal sci-fi background throughout the application with crisp, white text that is clearly readable against the dark theme, creating an immersive visual experience.
